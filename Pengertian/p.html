KERNEL

Kernel adalah suatu program yang menghubungkan antara software dan hardware. Kernel juga mengatur dan mengelola hardware. Kernel mempunyai banyak jenis. Di android ada 3 jenis kernel yang sering digunakan, seperti:

  1. Android Common Kernel (ACK)
  2. Kernel AOSP
  3. Kernel GKI

UI

UI(User Interface)adalah jalan untuk user berinteraksi dengan komputer.

OS

OS adalah sistem operasi yang mengelola perangkat keras dan perangkat lunak pada komputer atau perangkat lainnya.OS punya banyak jenisnya,contoh OS di HP yaitu:IOS,Android,linux
